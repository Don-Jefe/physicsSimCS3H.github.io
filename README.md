# AP Physics Demos — Mechanics

This small site contains interactive demos for AP Physics mechanics unit implemented in plain JavaScript and HTML/CSS.

Features:
- Projectile Motion: adjust angle, speed, gravity and launch a projectile.
- Simple Harmonic Motion: mass-spring demo with damping.
- Electric Circuit (RC): simulate capacitor charging/discharging with adjustable R, C, and Vin.

How to run:
1. Open `index.html` in your browser (double-click or serve with a simple HTTP server).

Quick tips:
- Use the navigation links to switch demos.
- Adjust sliders to see how parameters affect motion and graphs.
